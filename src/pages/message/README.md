# Message 消息提示

窗口顶部显示，用于系统提醒等通知。

## 使用说明

需要使用 [<ComponentProvider>](./component-provider) 包裹页面根组件。

## 使用示例

[Code](./demo/index.tsx)

## API

- `Message.info(options)`

- `Message.success(options)`

- `Message.warning(options)`

- `Message.error(options)`

- `Message.hide()`
